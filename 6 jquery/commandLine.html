Various Commands

  1. LS --> LIST
  2. CD --> CHANGE DIRECTORY
  3. CD.. --> BACK TO PREVIOS DIRECTORY
  4. ALT --> TO GO BETWEEN THE GIVEN PATH
  5. CTRL E AND CTRL U --> GO AT LAST & CLEAR CMD
    6. pwd
  6. MKDIR NAME OF DIR --> TO MAKE FOLDER
  7. TOUCH --> FOR NEW FILE CREATION
  8. START FILENAME --> TO OPEN FILE
  9. CODE FILENAME --> TO OPEN FILE
  10. RM FILENAME --> DELETE FILE
  11. RM * --> DELETE all FILE
  12. RM -r filename--> DELETE folder
  13. pwd --> current location
